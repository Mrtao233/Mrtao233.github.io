b站视频放到markdown

发表于 2023-09-09 更新于 2023-09-17 分类于 hexo

  1. 在b站视频>分享>嵌入代码,得到iframe标签

  2. 将其复制到markdown中

  3. iframe标签略做修改,src中加上https:, 也可以改宽度,width: 1000px; height: 500px;

  4. 再url后面加上&autoplay=0可以关闭自动播放
